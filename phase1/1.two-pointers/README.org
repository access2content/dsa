** Phase 4 — Advanced Pointer Semantics [1/3]
Pointers are no longer just moving through data. They now track invisible state and evolving conditions. Learn that pointers can do more than traverse arrays. They can:
- Track changing validity
- Simulate hidden effects
- Reason using global boundaries
  
Core mechanics:
- Maintaining dynamically valid regions
- Skipping logically removed state
- Using global constraints to make local decisions

* How to solve
After every problem, write:
- Invariant - “What is always guaranteed to be correct?”
- Pointer movement rule - “What condition allows this pointer to move?”
- Why movement is safe - “Why can I permanently ignore the element(s) I just passed?”
