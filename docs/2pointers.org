* How to solve
After every problem, write:
- Invariant - “What is always guaranteed to be correct?”
- Pointer movement rule - “What condition allows this pointer to move?”
- Why movement is safe - “Why can I permanently ignore the element(s) I just passed?”

Example for 167:
Invariant:
- Array is sorted.

Movement rule:
- If sum too small → move left.
- If sum too large → move right.

Why safe:
- Sorted order guarantees discarded pairs
- cannot become valid later.
